# OS-LEVEL DEPENDENCIES (not pip installable):
# ┌─────────────────────────────────────────────────────────
# │ Windows: Install Npcap for live packet capture
# │   → https://npcap.com/#download
# │   → Run as Administrator, enable WinPcap-compatible mode
# │
# │ Linux:   Install libpcap for live packet capture
# │   Ubuntu/Debian: sudo apt install -y libpcap-dev tcpdump
# │   Fedora/RHEL:   sudo dnf install -y libpcap-devel tcpdump
# │   Arch:          sudo pacman -S libpcap
# │   OR (no sudo needed after):
# │     sudo setcap cap_net_raw+eip $(which python3)
# │
# │ macOS:   libpcap is built-in — no install needed
# │   Run with: sudo ./start.sh
# │   OR:       sudo chmod +r /dev/bpf*  (until reboot)
# └─────────────────────────────────────────────────────────
#
# SCREEN CAPTURE (for Video Call Monitor):
#   All OS:  pip install mss         ← recommended
#   Linux:   sudo apt install scrot  ← fallback
#   macOS:   System Settings → Privacy → Screen Recording → enable Terminal
#   Windows: Works automatically
